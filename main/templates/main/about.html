Information about the project and me

This web blog was built on the basis of the Django framework, which works with the database and has the functionality to create, edit and delete articles

This is my first project made with Django and if you are interested, you can check out the open source code at GitHub. You can also find there my two previous projects, in one of which I passed the parsed data from a third-party API to the database using MySQL and with the help of the Tornado-web framework rendered them on the site with the ability to specify the required date range. In the second project I created a web-blog based on the Flask framework, where you can also add, edit and delete articles.

The next step for me will be to creation an online store based on the Django framework with the possibility of authorization and user registration (model - User and Customer), leave comments, making payments online and creating personal store API.

In the near future, I see myself as I am a strong Middle Python Developer and I think that in order to rise I need real practice, complete real tasks. So I will leave a link to my CV through which you can contact me.
